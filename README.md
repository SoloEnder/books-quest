## Introduction

Reading is great, and so is playing. So why not combine the two? That's where Book Quest comes in.

*You read, you play.*
Book Quest uses video game mechanics to give you the satisfaction of completing quests, defeating monsters, or simply leveling up… all while reading.

Please note that the project is still under development; therefore, many of the features many of the features described above have not been implemented yet.

## Installation
Unfortunately, Book Quest is currently only available for Windows or by building it from source.

### Windows
- Go to the [official releases page](https://github.com/SoloEnder/books-quest/releases), navigate to the latest release, and follow the instructions provided there.

### Building from source code
- You must have Python 3.13.7 or later installed (an earlier version might work, but this has not been tested).
- Clone this repository (or download the source code):
```bash
git clone https://github.com/SoloEnder/books-quest.git
```
- Create and activate a Python virtual environment (optional, but recommended).
- Install the project dependencies:
```bash
pip install -r requirements.txt
```
- Then, run the `main.py` file:
```bash
py main.py
```
